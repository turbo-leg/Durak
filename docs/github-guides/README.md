## **🎯 Quick Commands**

### **First Time Setup**

```bash
gh auth login
```

### **See All Issues**

```bash
gh issue list
```

### **View Specific Issue**

```bash
gh issue view 5
```

### **Start Working on Issue #5**

```bash
gh issue develop 5
```

### **See Changes You Made**

```bash
git diff
```

### **Save Changes (Commit)**

```bash
git add .
git commit -m "what you changed"
```

### **Upload Changes (Push)**

```bash
git push
```

### **Create Pull Request**

```bash
gh pr create
```
